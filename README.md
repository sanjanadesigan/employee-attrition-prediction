# Employee Attrition Prediction
A machine learning project that predicts employee attrition using the **IBM HR Analytics Employee Attrition Dataset**.
The objective of this project is to identify employees at risk of leaving by analyzing workplace, demographic, and job-related factors. 
The workflow includes **data cleaning, exploratory data analysis (EDA), feature engineering, preprocessing, and Logistic Regression modeling**.
To improve real-world usefulness, the model was optimized for **high recall**, prioritizing the identification of employees likely to leave over raw accuracy.

### Key Highlights
* Exploratory Data Analysis (EDA)
* Feature Engineering & Preprocessing
* Logistic Regression for Classification
* Handling Class Imbalance
* Threshold Tuning for Improved Recall
* Final Model Recall: **82%**

### Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
